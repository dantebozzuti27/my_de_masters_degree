## How to Complete a Lesson

1. Navigate to the lesson file: `workspace/monthX/week-XX/exercises/dayXX_*.py`
2. Read the docstring and exercises
3. Complete all exercises
4. Delete the `# YOUR CODE HERE` marker
5. Run the file to verify: `python3 dayXX_*.py`
6. Scanner will detect completion at next build
